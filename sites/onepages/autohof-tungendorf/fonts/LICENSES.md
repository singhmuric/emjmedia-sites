# Self-hosted Webfonts — Lizenzen

Stand: 2026-04-21 (Session 1.3 / T-044)

Alle drei Webfonts in diesem Ordner stehen unter der **SIL Open Font License 1.1** und dürfen ohne Lizenzgebühren kommerziell genutzt, gehostet und in Webseiten eingebettet werden, solange die Lizenz selbst zugänglich bleibt.

Constitution §1.4 erlaubt max. 2 selbst gehostete Webfonts pro Site. Pro Variant:
- **a (Meister klassisch):** Fraunces (heading) + Inter (body) = 2 Webfonts
- **b (Moderne Werkstatt):** Inter durchgehend = 1 Webfont
- **c (Handwerk pur):** JetBrains Mono (heading) + Inter (body) = 2 Webfonts

## Quellen

- **Fraunces** — Undercase Type, https://fonts.google.com/specimen/Fraunces — SIL OFL 1.1
- **Inter** — Rasmus Andersson, https://rsms.me/inter/ — SIL OFL 1.1
- **JetBrains Mono** — JetBrains s.r.o., https://www.jetbrains.com/lp/mono/ — SIL OFL 1.1

Die WOFF2-Subsets (`*-latin.woff2`, `*-latin-ext.woff2`) wurden aus den entsprechenden npm-Paketen `@fontsource-variable/{fraunces,inter,jetbrains-mono}` extrahiert. Die Fontsource-Pakete pflegen die Subsets gegen die offiziellen Quellen.

## SIL OFL 1.1 — Pflichten

1. Die Lizenz muss mit der Software/Bundle ausgeliefert werden — diese Datei erfüllt das.
2. Der Name der Schriftart darf **nicht** für abgeleitete Werke wiederverwendet werden, wenn das Original modifiziert wird (Reserved Font Name).
3. Die Schriftart darf weiterverkauft werden — aber **nicht** allein als Verkaufsobjekt; sie muss Teil eines größeren Werks sein.

Volltext der OFL: https://openfontlicense.org/open-font-license-official-text/
